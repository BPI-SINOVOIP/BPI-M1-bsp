Welcome to the Nanoradio NRX API Reference Documentaion

This reference documentation provides details about the Nanoradio NRX API, an easy-to-use application interface that provides access to most standard WLAN functions and Nanoradio specific functions of the Nanoradio WLAN Driver.

The Nanoradio WLAN Driver is used together with the Nanoradio series of Wi-Fi/WLAN 802.11b/g chipsets and SIP modules, designed to bring outstanding WLAN capabilities into mobile phones and other battery powered consumer devices.

 

Purpose and intended audience

This guide is intended for experienced programmers who are developing host applications or embedded applications that are going to be used in a Wi-Fi target device based on the NRX700 chipset or module. It is intended to be a complete reference guide for software development using the Nanoradio NRX API.

You should have a working knowledge of the following:

•          Wi-Fi/WLAN standards and protocols

•          The C or Embedded C++ programming languages.

•          The architecture of the NRX Host Environment.

•          The architecture and requirements of your host system.

•          The operating system and IDE of your development system.
